# 法律RAG测试题库

基于讯问笔录场景的测试题目，用于评测法律案件场景下的RAG系统质量。

## 📁 题库文件

```
benchmark/
├── 01-事实型题目.md       # 10题
├── 02-证据集合型题目.md    # 10题
└── 03-冲突缺口型题目.md   # 15题
```

**总计：35道测试题**

## 📋 题型说明

### 1. 事实型
测试提取唯一精确答案的能力。
- 金额：42000元总收款、18000元退款、24000元欠款
- 日期：2020-06-05、2020-08-31
- 个人信息：姓名、身份证号、电话
- 是/否问题：是否认罪认罚

### 2. 证据集合型
测试聚合多条证据的能力。
- 承诺退款/已退款/仍欠款的证据
- 资金用途说明的证据
- 微信转账记录
- 联系和寻找过程

### 3. 冲突缺口型
测试是否会在信息缺失时编造内容。
- "忘记了/不详/不知道"的信息应拒答
- 禁止幻觉和猜测

## 📄 样本数据

基于 `SampleData/output.md` 讯问笔录：
- 被讯问人：陈明飞
- 涉案金额：42000元（收款）→ 18000元（退款）→ 24000元（欠款）
- 时间范围：2020年6月-11月

## 💡 使用方式

将题目用于测试RAG系统在法律文档上的表现。
