# MVP功能清单

## 核心功能 (5个)

### F1: 案卷智能导入与解析
- 优先级: P0
- 状态: 待开发
- 描述: 支持PDF/图片/JSON导入, 自动识别文书类型, 提取元数据
- 验收标准: 识别准确率>90%, 导入时间<1分钟

### F2: 案件概览自动生成
- 优先级: P0
- 状态: 待开发
- 描述: 提取案件信息, 生成摘要, 时间线, 程序节点
- 验收标准: 信息提取准确率>95%

### F3: 证据自动分类与汇总
- 优先级: P0
- 状态: 待开发
- 描述: 自动分类7大证据类型, 统计数量, 导出清单
- 验收标准: 分类准确率>85%

### F4: 证据矛盾点识别
- 优先级: P0 (亮点)
- 状态: 待开发
- 描述: 自动比对证据, 识别矛盾, 生成报告
- 验收标准: 准确率>75%, 覆盖度>70%

### F5: 质证要点建议
- 优先级: P0 (亮点)
- 状态: 待开发
- 描述: 基于证据分析, 生成质证要点, 引用法律条文
- 验收标准: 相关性>80%, 条文准确率>90%

## 二期功能 (Phase 2)

### F6: 类案检索
- 优先级: P1
- 描述: 检索类似案件, 提供参考

### F7: 辩护策略建议
- 优先级: P1
- 描述: 基于案情, 提供辩护策略建议

### F8: 法律文书生成
- 优先级: P2
- 描述: 生成取保申请书、辩护词等文书
